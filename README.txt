Appendix W: Implementation Material

Implementation material is non-normative and includes schema, examples, WSDL, FHIR conformance resources, and more.

Implementation material for some IHE profiles can be found on IHE Google Drive https://drive.google.com/drive/folders/1aHW4ChzRzaYSoyewi9zGIRHwVphgXQst
IT Infrastructure Implementation Materials can be found in the IHE ITI GitHub repo https://github.com/IHE/ITI-Info

FHIR conformance resources (e.g., StructureDefinition, CapabilityStatement, ValueSet, etc) for some IHE profiles can be found on the IHE controlled “fhir” GitHub repository found at https://github.com/IHE/fhir.

Format Code vocabulary is published at http://profiles.ihe.net/fhir/ihe.formatcode.fhir/
IT-Infrastructure has many Implementation Guides at http://profiles.ihe.net/ITI
